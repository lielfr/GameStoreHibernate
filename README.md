# Game store using Hibernate
This is a companion repository to the Software Engineering course, Department of Computer Science, University of Haifa, Fall semester 2020-2021.
It contains a purposed solution to the game store exercise in lab 5.

## Running
1. Make sure the hibernate.properties file is set correctly and the target schema exists.
2. Run using the following goal chain (can be run together): `clean install exec:java`.

Good luck!
